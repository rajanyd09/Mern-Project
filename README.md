# Gemini Flow MERN Project

An AI-powered interactive flow canvas built with **React**, **React Flow**, **Node.js**, **Express**, and **MongoDB**, featuring **Gemini 2.5 Flash** for intelligent content generation.

## 🚀 Quick Start

1.  **Backend**: `cd backend && nodemon server.js` (uses nodemon)
2.  **Frontend**: `cd frontend && npm run dev` (uses vite)

## ✨ Features
-   **Interactive Canvas**: Drag and drop nodes to build your AI logic.
-   **Gemini AI Integration**: High-speed AI responses using the latest Flash models.
-   **Persistence**: Save your favorite AI interactions to MongoDB.
-   **Modern UI**: Sleek dark-mode compatible aesthetics with GSAP animations.
